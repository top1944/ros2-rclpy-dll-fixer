📋 使用前准备
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 打开命令提示符（cmd）或PowerShell

2. 设置ROS2_HOME环境变量：
   set ROS2_HOME=C:\dev\ros2_humble

   注意：如果您的ROS2安装在其他位置，请修改路径


🚀 快速修复（3步）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

方法1：双击运行（最简单）
  └─ fix_rclpy_dll_simple.bat

方法2：命令行运行
  └─ python 一键修复rclpy.py

方法3：验证工具状态
  └─ verify_tools.bat

⚠️ 重要提示
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 必须先设置 ROS2_HOME 环境变量
2. Python版本必须是 3.10
3. 系统必须是 64位 Windows
4. 如果修改了ROS2安装路径，需要重新设置ROS2_HOME
